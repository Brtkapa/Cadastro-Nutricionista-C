# Cadastro-Nutricionista-C
Sistema básico de Cadastro Nutricional em C. Utiliza struct para modelar o paciente (nome, idade, peso, altura) e implementa o cálculo do IMC (Índice de Massa Corporal). O projeto foca no uso de Estruturas, modularização (typedef e funções) e I/O para um CRUD simples em console.
